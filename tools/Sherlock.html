Sherlock Complete Cheatsheet

  • Clone repo: git clone https://github.com/sherlock-project/sherlock.git
  • Install dependencies: pip install -r requirements.txt
  • Run tool: python3 sherlock.py username
  • Supports Python 3.x
  • Search for a single username: python3 sherlock.py username
  • Search multiple usernames: python3 sherlock.py user1 user2 user3
  • Save output to file: python3 sherlock.py username -o output.txt
  • Use verbose output: --verbose
  • Searches 300+ social media and website platforms including:
  • Twitter, Instagram, Facebook, TikTok, Reddit, GitHub, LinkedIn, YouTube, Pinterest, Tumblr, and more
  • Constantly updated platform list via GitHub repo
  • --json — Output results in JSON format
  • --timeout <seconds> — Set network timeout (default 10s)
  • --proxy <proxy_url> — Route requests through a proxy
  • --tor — Route traffic through Tor network (requires Tor setup)
  • OSINT investigations to find social media profiles by username
  • Checking username availability across platforms
  • Recon for penetration testing and bug bounty programs
  • Verifying online presence and footprint of targets
  • Combine with grep or other tools for filtering
  • Use --timeout to avoid delays on slow networks
  • Keep Sherlock updated for newest platform checks
  • Use in scripts to automate OSINT pipelines
