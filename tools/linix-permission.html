Linux Cheatsheet - Permissions & Ownership

Change file or directory permissions.

ExampleDescription
chmod 755 fileOwner=rwx, Group=rx, Others=rx
chmod u+x script.shAdd execute permission to user
chmod -R 644 *Recursive change: read/write user, read others

Change file ownership (user and group).

Syntax: chown [user][:group] file

Examples:
chown alice file.txt
chown bob:dev file.txt
chown -R root:root /opt

Change the group of a file/directory.

Example: chgrp staff report.doc

Sets default permissions for new files.

Example: umask 022 → new files: 644, dirs: 755

Show file details, including permissions, owner, and timestamps.

Example: stat file.txt

Output:
