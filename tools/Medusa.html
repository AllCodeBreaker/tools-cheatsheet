Medusa Cheatsheet

  • Medusa is a fast, modular, parallel login brute-forcer.
  • Supports many protocols (SSH, FTP, HTTP, SMB, VNC, etc.)
  • Designed for speedy and efficient password brute forcing.
CommandDescription
medusa -h <host> -u <user> -P <passlist> -M sshBrute force SSH login for a single user
medusa -h <host> -U <userlist> -P <passlist> -M ftpBrute force FTP login for multiple users
medusa -H <hosts> -U <userlist> -P <passlist> -M httpBrute force HTTP login on multiple hosts
medusa -h <host> -u <user> -p <password> -M smbTest a single SMB login
medusa -L users.txt -P passwords.txt -M ssh -h 192.168.1.100Use user and password lists to brute force SSH
