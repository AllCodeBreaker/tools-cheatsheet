Linux Cheatsheet - Compression & Archiving

Archive files and directories into tarballs.

CommandDescription
tar -cvf archive.tar /pathCreate archive (verbose)
tar -xvf archive.tarExtract archive
tar -czvf archive.tar.gz /pathCreate gzip compressed tarball
tar -xvzf archive.tar.gzExtract gzip compressed tarball
tar -cJvf archive.tar.xz /pathCreate xz compressed tarball
tar -xvJf archive.tar.xzExtract xz compressed tarball

Flags: c=create, x=extract, v=verbose, f=file, z=gzip, J=xz

Compress or decompress files using gzip.

gzip file.txt → creates file.txt.gz

gunzip file.txt.gz → decompress

gzip -d file.txt.gz also decompresses

Zip compression tools.

CommandDescription
zip archive.zip file1 file2Create zip archive
unzip archive.zipExtract zip archive
unzip -l archive.zipList contents

Compress or decompress files using xz.

xz file.txt → compress to file.txt.xz

unxz file.txt.xz → decompress
