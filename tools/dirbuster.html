Dirb / Dirbuster Complete Cheatsheet

Dirb and Dirbuster are tools used for brute forcing directories and files on web servers to discover hidden or unlinked content.

  • Scan a URL with default wordlist:
    dirb http://example.com/
  • Use a custom wordlist:
    dirb http://example.com/ /path/to/wordlist.txt
  • Use HTTPS:
    dirb https://example.com/
  • Ignore response codes:
    dirb http://example.com/ -r
  • Run Dirbuster GUI:
    dirbuster
  • Use command line with options:
    dirbuster -u http://example.com -l /path/wordlist.txt -t 50
  • Specify extensions:
    -x php,html,txt
  • Set thread count:
    -t 50 (default 10)
FlagDescription
-rRecursive scan
-wUse wildcard wordlist
-oOutput results to file
-zDelay between requests
-sShow only responses with specific codes
  • Use large comprehensive wordlists like SecLists
  • Be cautious with threading to avoid server overload or IP bans
  • Combine with proxy tools for anonymity
  • Check server response codes to avoid false positives
