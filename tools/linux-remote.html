Linux Cheatsheet - SSH & Remote Access

CommandDescription
ssh user@hostConnect to remote host via SSH
ssh -p 2222 user@hostConnect using custom port 2222
ssh -i ~/.ssh/id_rsa user@hostUse specific private key file
ssh -v user@hostVerbose mode (debug connection)
ssh -L 8080:localhost:80 user@hostLocal port forwarding (e.g. access remote web server)
CommandDescription
scp file.txt user@host:/remote/pathCopy local file to remote
scp user@host:/remote/file.txt ./Copy remote file to local
scp -P 2222 file.txt user@host:/pathSpecify remote SSH port
scp -r folder user@host:/remote/Copy folder recursively

Interactive SSH file transfer client.

CommandDescription
sftp user@hostStart SFTP session
put localfileUpload file
get remotefileDownload file
lsList remote directory
cd remote_dirChange remote directory
exitExit SFTP
CommandDescription
ssh-keygen -t rsa -b 4096Generate RSA key pair with 4096 bits
ssh-keygen -t ed25519Generate Ed25519 key pair (recommended)
ssh-copy-id user@hostCopy public key to remote host for passwordless login
