Dnsenum / Dnsrecon Complete Cheatsheet

  • Enumerate domain:
    dnsenum example.com
  • Use alternative DNS server:
    dnsenum --dnsserver 8.8.8.8 example.com
  • Use wordlist for subdomains:
    dnsenum --wordlist subdomains.txt example.com
  • Save output:
    dnsenum --output results.txt example.com
  • Standard enumeration:
    dnsrecon -d example.com
  • Zone transfer check:
    dnsrecon -d example.com -t axfr
  • Brute force subdomains:
    dnsrecon -d example.com -D subdomains.txt -t brt
  • Perform all tests:
    dnsrecon -d example.com -a
  • Output to file:
    dnsrecon -d example.com -a -o output.xml -f xml
