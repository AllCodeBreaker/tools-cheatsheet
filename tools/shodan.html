Shodan Complete Cheatsheet

  • Sign up at shodan.io for API key
  • Install CLI: pip install shodan
  • Initialize CLI with API key: shodan init YOUR_API_KEY
CommandDescription
shodan search "apache"Search devices running Apache
shodan host 8.8.8.8Get info about an IP address
shodan scan submit 1.2.3.4/24Start a scan on subnet
shodan stats apacheGet stats on query
shodan infoShow account information
  • port: Search by port number (e.g. port:22)
  • org: Search by organization (e.g. org:"Google LLC")
  • country: Filter by country (e.g. country:US)
  • hostname: Search by hostname
  • before/after: Search by date (e.g. before:2024-01-01)
  • Combine filters for precise results (e.g. apache port:80 country:US)
import shodan

API_KEY = "YOUR_API_KEY"
api = shodan.Shodan(API_KEY)

try:
    results = api.search('apache')
    print(f"Results found: {results['total']}")
    for result in results['matches'][:5]:
        print(result['ip_str'])
except shodan.APIError as e:
    print(f"Error: {e}")
