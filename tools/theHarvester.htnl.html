theHarvester Complete Cheatsheet

theHarvester -d <domain> -b <source> [options]

Example:
theHarvester -d example.com -b google
SourceDescription
googleGoogle Search
bingBing Search
crtshCertificate Transparency Logs
linkedinLinkedIn (with API)
githubGitHub User Discovery
hunterHunter.io (API key required)
shodanHost Enumeration (API key)
anubisSubdomain enumeration
allUse all available sources
-d <domain> — Target domain
-b <source> — Data source to use
-l <limit> — Limit number of results
-s <start> — Result start number (pagination)
-f — Save output to HTML and XML
-v — Verbose output
-h — Show help menu
-f <filename> — Export to HTML and XML
Output files: filename.xml, filename.html
Supports CLI parsing or browser review
Edit API keys in:
theHarvester/api-keys.yaml

Example format:
shodan:
  api_key: YOUR_KEY_HERE

hunter:
  api_key: YOUR_KEY_HERE
🔎 **Search domain using Google**:
theHarvester -d example.com -b google

📦 **Run all sources, limit results, export to HTML**:
theHarvester -d example.com -b all -l 100 -f output

🔐 **Use with API services (Shodan, Hunter.io)**:
theHarvester -d example.com -b shodan
