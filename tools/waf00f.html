Waf00f Cheatsheet

  • Waf00f is a tool to detect and identify Web Application Firewalls (WAFs) protecting a web application.
  • Supports detection of many popular WAFs like ModSecurity, Cloudflare, AWS WAF, etc.
  • Useful during penetration testing to identify WAF protections and bypass techniques.
CommandDescription
wafw00f http://example.comDetect WAF on the specified URL
wafw00f --helpShow help and options
wafw00f -a http://example.comPerform aggressive detection
wafw00f -v http://example.comVerbose output
wafw00f --list-wafsList all supported WAFs
  • Run Waf00f early during reconnaissance to adjust your testing strategy.
  • Combine with other tools like Burp Suite for WAF bypass techniques.
  • Check multiple URLs and parameters to confirm WAF detection.
  • Use aggressive mode cautiously, as it might trigger alerts.
