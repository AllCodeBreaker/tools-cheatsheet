Linux Cheatsheet - System Information

Displays system information like kernel name, version, and architecture.

OptionDescription
-aAll system info
-sKernel name
-rKernel release
-mMachine hardware name

Example: uname -a

Output:
Linux ubuntu 5.15.0-75-generic #82-Ubuntu SMP x86_64 GNU/Linux

Shows or sets the system's hostname.

OptionDescription
(none)Print current hostname
-fFully qualified domain name
-iIP address

Example: hostname -f

Output:
myhost.localdomain

Displays how long the system has been running.

Example: uptime

Output:
14:33:02 up 2 days, 3:10, 3 users, load average: 0.25, 0.40, 0.35

Prints the current logged-in username.

Example: whoami

Output:
john

Displays user ID (UID), group ID (GID), and group memberships.

Example: id

Output:
uid=1000(john) gid=1000(john) groups=1000(john),27(sudo)

Interactive real-time view of running processes and system resource usage.

Example: top

Tip: Press q to quit.

Shows memory usage statistics.

OptionDescription
-hHuman-readable format (MB, GB)
-tShow total memory usage

Example: free -h

Output:
Mem: 7.8Gi used 1.2Gi free 6.6Gi

Shows disk space usage of mounted filesystems.

OptionDescription
-hHuman-readable format
-TShow filesystem types

Example: df -h

Shows size of directories and their contents.

OptionDescription
-shSummarize total size in human-readable format
-aShow size of all files

Example: du -sh *
