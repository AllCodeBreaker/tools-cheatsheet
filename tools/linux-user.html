Linux Cheatsheet - User Management

who: Lists logged-in users

whoami: Shows current user

Example: who

Output:
john tty7 2025-08-19 09:34

Shows UID, GID, and group membership of the current user.

Example: id

Output:
uid=1000(john) gid=1000(john) groups=1000(john),27(sudo)

users: List of logged-in users

groups: Groups the current user belongs to

Example: groups
john sudo docker

Creates a new user.

adduser is more user-friendly; useradd is lower-level.

Example: sudo adduser alice

Example: sudo useradd -m -s /bin/bash bob

Sets or changes a user's password.

Example: sudo passwd alice

Modifies user properties (username, shell, group, etc.)

OptionDescription
-aGAdd user to group(s)
-sChange login shell
-lChange username

Example: sudo usermod -aG docker alice

Create or modify a user group.

Example: sudo groupadd developers

Rename: sudo groupmod -n devs developers

Administers group password or membership.

Example: sudo gpasswd -a alice docker

Remove user or group from the system.

Delete user: sudo deluser alice

Delete group: sudo delgroup devs
