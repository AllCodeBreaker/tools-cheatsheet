Wireshark Cheatsheet

  • Wireshark is a widely used network protocol analyzer.
  • Allows capturing and interactive browsing of network traffic.
  • Supports deep inspection of hundreds of protocols.
ActionDescription
Start captureSelect interface & click Start
Stop captureClick the red square button
Open capture fileFile → Open or Ctrl+O
Save captureFile → Save or Ctrl+S
Apply display filterUse filter box to narrow traffic
FilterDescription
ip.addr == 192.168.1.1Show packets from/to IP
tcp.port == 80Filter HTTP traffic
udpShow UDP packets
http.requestShow HTTP requests
dnsShow DNS packets
tcp.flags.syn == 1 and tcp.flags.ack == 0Show SYN packets (TCP connection starts)
  • Use color rules to highlight interesting packets.
  • Save filtered packets for focused analysis.
  • Use follow TCP stream feature to view conversations.
  • Export objects (images, files) from capture for analysis.
  • Use Wireshark command line (tshark) for scripting.
