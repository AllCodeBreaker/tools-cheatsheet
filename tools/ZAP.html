ZAP Proxy Complete Cheatsheet

  • Download and run OWASP ZAP from https://www.zaproxy.org
  • Configure browser proxy to 127.0.0.1:8080 to intercept HTTP/S traffic
  • Use ZAP’s embedded browser or install root CA certificate for HTTPS interception
FeatureDescription
ProxyIntercept and modify HTTP/S requests and responses
SpiderAutomated crawling to map the target website
Active ScanAutomated vulnerability scanner
Passive ScanBackground scan on proxied traffic
FuzzerAutomated fuzz testing of input points
Forced BrowseDiscover hidden files and directories
Script ConsoleRun custom scripts for automation or scans
  • Enable/disable intercept via Proxy → Intercept
  • View and modify requests in the History tab
  • Right-click requests to send to other tools: Spider, Active Scan, Fuzzer
  • Set target scope under Scope to focus scans
  • Spider: Crawl site for pages and endpoints
  • Active Scan: Perform automated vulnerability scanning on target
  • Passive Scan: Analyze proxied traffic for common issues
  • Fuzzer: Test input points with custom payloads
  • Forced Browse: Find hidden files/directories
  • Marketplace: Browse and install add-ons from within ZAP
  • API Scanner: Scan REST/GraphQL APIs
  • Authentication Helpers: Support for complex login workflows
  • Plug-n-Hack: Integrate with browsers and tools easily
  • Manage authentication and session tokens under Session Properties
  • Configure context-specific authentication
  • Use scripts/macros to automate login and session maintenance
ShortcutAction

Ctrl + Shift + SStart Active Scan
