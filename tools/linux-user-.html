Linux Cheatsheet - User & Permission Management

CommandDescription
sudo adduser usernameCreate new user interactively
sudo useradd -m usernameCreate new user with home directory
sudo passwd usernameSet or change user password
sudo userdel -r usernameDelete user and home directory
id usernameShow user UID, GID, and groups
whoamiPrint current username
CommandDescription
sudo groupadd groupnameCreate a new group
sudo usermod -aG groupname usernameAdd user to group
groups usernameShow groups of a user
getent group groupnameShow group info
CommandDescription
ls -l filenameShow file permissions and ownership
chmod 755 filenameSet permissions (rwxr-xr-x)
chmod u+x filenameAdd execute permission for user
chown user:group filenameChange ownership
stat filenameShow detailed file info

CommandDescription
