OpenVAS Cheatsheet

  • OpenVAS is an open-source vulnerability scanner.
  • Includes a regularly updated feed of Network Vulnerability Tests (NVTs).
  • Used for vulnerability assessment and management.
  • Comprises several components including scanner, manager, and web interface.
CommandDescription
openvas-setupInstall and setup OpenVAS (deprecated in newer versions)
greenbone-nvt-syncSynchronize vulnerability tests
openvas-startStart OpenVAS services (older)
gvm-manage-certs -aGenerate SSL certificates
gvm-cliCommand line interface to interact with OpenVAS/GVM
  • Manage targets, tasks, and reports via gvmd or web UI (GSA).
  • Scan configuration includes target hosts and scan settings.
  • Run scans and generate reports via the web interface or CLI.
  • Update NVT feeds regularly for accurate detection.
  • Run greenbone-nvt-sync frequently to keep plugins updated.
  • Scan in segments to avoid network overload.
  • Use credentialed scans for deeper vulnerability assessment.
  • Export reports in PDF, XML, or CSV for analysis.
  • Regularly backup OpenVAS/GVM data and configs.
