Sublist3r Complete Cheatsheet

  • Clone repo: git clone https://github.com/aboul3la/Sublist3r.git
  • Install dependencies: pip install -r requirements.txt
  • Run tool: python sublist3r.py -d example.com
  • Enumerate subdomains: python sublist3r.py -d example.com
  • Save results to file: python sublist3r.py -d example.com -o subs.txt
  • Use multiple threads: -t 50 (default 10)
  • Verbose output: -v
FlagDescription
-dDomain name to enumerate
-oOutput file
-tNumber of threads
-vVerbose output
-eUse search engines (Google, Bing, Baidu, Yahoo, Ask, Netcraft, DNSdumpster, Virustotal)
-bSpecify which search engines (comma separated)
-pPort scan discovered hosts (requires nmap)

  • Use port scanning only if legally permitted
  • Combine with other tools for deeper enumeration
  • Adjust thread count to avoid rate-limiting
  • Use output files for automated processing
