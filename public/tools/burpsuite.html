Burp Suite Complete Cheatsheet

  • java -jar burpsuite_community.jar — Start Burp Suite Community Edition
  • Configure browser proxy to 127.0.0.1:8080 to intercept traffic
  • Use Burp’s embedded browser for automatic proxy config
ToolDescription
ProxyIntercept, inspect, and modify HTTP/S traffic
SpiderAutomated crawling to map target website
Scanner (Pro)Automated vulnerability scanning
IntruderCustomizable automated attacks (e.g. fuzzing)
RepeaterManual crafting and resending of requests
SequencerAnalyze randomness of tokens and session IDs
DecoderEncode/decode data in multiple formats
ComparerCompare requests/responses side-by-side
  • Turn intercept On/Off via Proxy → Intercept
  • Use HTTP history tab to view logged requests
  • Right-click requests to send to Repeater, Intruder, or Comparer
  • Set scope to focus on specific hosts under Target → Scope
