Nmap Complete Cheatsheet

nmap [options] <target>
Examples:
nmap 192.168.1.1
nmap scanme.nmap.org
OptionDescription
192.168.1.1Single IP
192.168.1.1-10IP Range
192.168.1.0/24CIDR Notation
-iL list.txtInput from file
-iR 10Random 10 hosts
-6Enable IPv6
OptionDescription
-sSTCP SYN scan (default)
-sTTCP Connect scan
-sUUDP scan
-sAACK scan
-sXXmas scan
-p 80 — Scan specific port
-p 1-1000 — Scan port range
-p- — Scan all ports
--top-ports 100 — Top 100 common ports
-T0 to -T5 — Timing template
--min-rate / --max-rate — Rate control
--max-retries — Limit retries
--host-timeout — Timeout per host
-sV — Version detection
-O — OS detection
-A — Aggressive: OS, version, scripts
-sC — Default scripts
--script vuln — Vulnerability scripts
--script-args='var=value' — Script args
-f — Fragment packets
-D RND:10 — Decoys
-S IP — Spoof source IP
--data-length — Pad data
-oN — Normal
-oX — XML
-oG — Grepable
-oA — All formats
-sn — Ping scan
-Pn — Treat as alive
-PS, -PA, -PU — TCP/UDP/ACK pings
--traceroute — Trace route
--reason — Show reasons
--open — Only open ports
--iflist — List interfaces
nmap -T4 -F target — Fast scan
nmap -sS -sU -p T:1-65535,U:1-1000 target — Full TCP & UDP scan
nmap -A -oA results target — Aggressive + All output formats
