Linux Cheatsheet - Process Management

Displays active processes.

OptionDescription
auxAll users, show processes in detail
-efFull-format list of all processes

Example: ps aux | grep nginx

Interactive process viewer (real-time CPU/memory usage).

Example: top

Improved version of top, with better UI (requires install).

Example: htop

Install: sudo apt install htop

kill: Terminates a process by PID.

killall: Terminates all processes by name.

SignalEffect
-9Force kill (SIGKILL)
-15Graceful kill (SIGTERM)

Examples:
kill -9 1234
killall firefox

Set or adjust priority of processes.

CommandDescription
nice -n 10 script.shStart with lower priority
renice -n -5 -p 1234Increase priority of PID 1234

Manage background & foreground jobs in a shell.

CommandDescription
jobsList background jobs
fg %1Bring job #1 to foreground
bg %1Resume job #1 in background

Example: sleep 60 & → jobs → fg %1
