Hashcat Cheatsheet

  • Hashcat is a powerful password recovery tool supporting GPU acceleration.
  • Supports many hash types (MD5, SHA, NTLM, WPA, etc.)
  • Commonly used for brute-force, dictionary, combinator, mask, and hybrid attacks.
CommandDescription
hashcat -m 0 -a 0 hash.txt wordlist.txtAttack MD5 hashes (mode 0) with dictionary attack
hashcat -m 1000 -a 3 hash.txt ?d?d?d?d?d?dBrute-force NTLM (mode 1000) with 6 digit mask
hashcat -bBenchmark supported algorithms
hashcat -m 2500 -a 0 hash.txt wordlist.txtAttack WPA/WPA2 handshakes
hashcat --show hash.txtShow cracked passwords from previous session

  • Use masks and rules to reduce the keyspace and speed up cracking.
  • Run hashcat -b first to benchmark your GPU.
  • Combine dictionary and brute-force attacks for best results.
  • Keep GPU drivers updated for maximum performance.
  • Always use hashcat legally and ethically.
