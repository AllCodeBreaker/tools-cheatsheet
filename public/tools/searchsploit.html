Searchsploit / ExploitDB Cheatsheet

  • Searchsploit is a command-line search tool for Exploit Database (ExploitDB).
  • Helps quickly find exploits, shellcode, and papers related to vulnerabilities.
  • Works offline after updating the local ExploitDB repository.
CommandDescription
searchsploit <search_term>Search ExploitDB for a keyword
searchsploit -w <search_term>Search with web links included
searchsploit -t <search_term>Search only titles
searchsploit --nmap <nmap_xml_file>Search based on Nmap XML output
searchsploit -m <exploit_path>Copy exploit to current directory
searchsploit --updateUpdate local ExploitDB database
  • searchsploit vsftpd — Search exploits related to vsftpd.
  • searchsploit -w apache 2.4 — Search for Apache 2.4 with web links.
  • searchsploit --nmap scan.xml — Find exploits based on Nmap scan results.
  • searchsploit -m exploits/linux/local/12345.c — Copy exploit locally.
  • Update the ExploitDB database regularly using searchsploit --update.
  • Use specific keywords to narrow down search results.
  • Cross-reference exploit details with CVE information.
  • Always test exploits in a safe, isolated environment.
