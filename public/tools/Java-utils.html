Java - java.util

java.util package provides a large collection of utility classes including data structures, date/time facilities, random number generation, and concurrency utilities. It is fundamental for Java development to manage collections and perform common programming tasks.

Classes

ArrayList

Purpose: Resizable-array implementation of the List interface. Provides dynamic arrays that grow as needed.

Constructors

ArrayList()

Creates an empty list with default capacity.

ArrayList(Collection c)

Creates a list containing elements of specified collection.

ArrayList(int initialCapacity)

Creates an empty list with specified initial capacity.

Methods

Method Signature Description Return Type Parameters
add boolean add(E e) Adds element at the end. boolean e - element to add
get E get(int index) Returns element at specified index. E index - position to fetch
remove E remove(int index) Removes element at index and returns it. E index - position to remove
size int size() Returns number of elements. int None

Best Practices

  • Use when frequent random access is needed.
  • Not optimized for frequent insertions/removals in middle (use LinkedList then).
  • Initial capacity tuning improves performance by reducing resizing.

HashMap

Purpose: Implementation of the Map interface based on hash table. Allows null keys and values, unordered.

Constructors

HashMap()

Constructs empty map with default capacity and load factor.

HashMap(int initialCapacity)

Constructs with specified initial capacity.

HashMap(int initialCapacity, float loadFactor)

Constructs with specified initial capacity and load factor.

HashMap(Map m)

Constructs map with mappings from specified map.

Methods

Method Signature Description Return Type Parameters
put V put(K key, V value) Associates specified value with key. V (previous value or null) key - mapping key, value - mapped value
get V get(Object key) Returns value to which key maps, or null. V key - lookup key
remove V remove(Object key) Removes mapping for key if present. V key - key to remove
containsKey boolean containsKey(Object key) Returns true if map contains key. boolean key - lookup key

Best Practices

  • Use for fast key-based lookup.
  • Not thread-safe; use ConcurrentHashMap for concurrency.
  • Initial capacity and load factor tuning can improve performance and memory usage.

LinkedList

Purpose: Doubly-linked list implementation of the List and Deque interfaces. Good for frequent insertions/removals.

Constructors

LinkedList()

Creates empty linked list.

LinkedList(Collection c)

Creates list containing elements of collection.

Methods

Method Signature Description Return Type Parameters
addFirst void addFirst(E e) Adds element at the start. void e - element to add
addLast void addLast(E e) Adds element at the end. void e - element to add
removeFirst E removeFirst() Removes and returns first element. E None
get E get(int index) Returns element at position. E index - position

Best Practices

  • Prefer LinkedList if your use case requires lots of insertions/removals.
  • Not ideal for frequent random access (slow get(index)).

Functions

Note: java.util is a package with mostly classes and interfaces; standalone utility functions are inside classes like Collections, Arrays.

Collections.sort

void sort(List list)

Sorts the specified list into ascending order according to natural ordering.

Parameters:

  • list - the list to be sorted

Returns: void

Failure: Throws ClassCastException if list elements are not mutually comparable.

Best Practices: Use when sorting Lists; prefer streams for functional style sorting in Java 8+.

Collections.shuffle

void shuffle(List list)

Randomly permutes the elements in the specified list.

Parameters:

  • list - the list to shuffle

Returns: void

Best Practices: Use for randomizing elements, e.g., in games or tests.

Arrays.asList

List asList(T... a)

Returns a fixed-size list backed by the specified array.

Parameters:

  • a - array to be converted

Returns: List view of the array.

Best Practices: Useful for quick conversion, but the returned list is fixed-size (cannot add/remove).
