Recon-ng Complete Cheatsheet

recon-ng — Start the framework
help — Show general help
exit — Quit the framework
workspaces — Show workspace commands
workspaces add <name> — Create a new workspace
workspaces select <name> — Switch to a workspace
workspaces delete <name> — Delete a workspace
workspaces list — List all workspaces
modules search <keyword> — Search modules
modules load <module/path> — Load a module
show options — View module options
set <option> <value> — Set option
run — Execute the module
CategoryPurpose
recon/domains-contactsEmail & contact discovery
recon/domains-hostsFind subdomains, hosts
recon/hosts-hostsExpand hostnames
recon/hosts-portsPort scanning modules
recon/hosts-vulnerabilitiesVulnerability discovery
show schema — View DB schema
show hosts, show domains, show contacts
delete from <table> — Remove data
export csv /path/file.csv — Export data
keys list — Show all stored API keys
keys add <service> <key> — Add a new API key
keys remove <service> — Remove an API key
marketplace search <term> — Search modules
marketplace install <name> — Install module
marketplace remove <name> — Remove module
use <module> — Alias for loading module
info — Show info about loaded module
clear — Clear screen
back — Return to main menu

run
