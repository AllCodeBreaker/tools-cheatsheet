Masscan Cheatsheet

masscan <target> -p<ports> --rate <pps>

Scan target IP or network for open ports at a given packet rate.

OptionDescription
-p<ports>Ports or port ranges to scan (e.g. 80,443 or 1-1000)
--rate <pps>Packets per second to send (e.g. 1000)
-iL <file>Input list of IPs or networks
-oX <file>Output results in XML format
-oG <file>Output results in grepable format
-oJ <file>Output results in JSON format
--interface <iface>Specify network interface
--source-ip <IP>Use custom source IP address
  • Scan ports 80 and 443 on 192.168.1.1 at 1000 pps:
    masscan 192.168.1.1 -p80,443 --rate 1000
  • Scan entire subnet for ports 1-1000:
    masscan 192.168.1.0/24 -p1-1000 --rate 500
  • Scan IPs from a file:
    masscan -iL targets.txt -p22,80 --rate 2000
  • Save results in XML:
    masscan 10.0.0.0/8 -p80 --rate 1000 -oX results.xml
  • Masscan is extremely fast; adjust --rate carefully to avoid flooding the network.
  • Run with root privileges for raw packet sending.
  • Use smaller subnets for better control and reliability.
  • Combine with Nmap for detailed scans of discovered ports.
