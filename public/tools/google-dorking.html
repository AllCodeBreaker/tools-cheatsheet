Google Dorking Cheatsheet

OperatorUsageExample
site:Search within a domainsite:example.com
filetype:Search for specific file typesfiletype:pdf
inurl:Find URLs containing a keywordinurl:admin
intitle:Find pages with keyword in titleintitle:"login page"
cache:View cached version of a pagecache:example.com
allintext:Find pages with all words in textallintext:password
  • filetype:sql "password" — Find SQL files with password data
  • intitle:"index of" passwd — Directory listings containing passwd files
  • inurl:wp-admin — WordPress admin pages
  • site:gov confidential — Confidential data on government sites
  • "error log" filetype:log — Exposed error logs
  • ext:env DB_PASSWORD — Environment files containing database passwords
  • Use quotes for exact phrase matching.
  • Combine operators for precise results.
  • Be ethical and avoid accessing unauthorized data.
  • Google may block or throttle queries; respect limits.
  • Use dorking as part of open-source intelligence (OSINT) investigations.
