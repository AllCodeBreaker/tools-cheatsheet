John the Ripper Cheatsheet

john <hashfile>

Crack password hashes from a file using default mode.

OptionDescription
--wordlist=<file>Use a custom wordlist for cracking
--rulesApply word mangling rules to wordlist
--format=<format>Specify hash type (e.g. md5, nt, sha512)
--showShow cracked passwords
--session=<name>Save session state for resuming later
--incrementalUse brute-force incremental mode
--mask=<pattern>Mask attack with custom charset
  • Crack hashes with default mode:
    john hashes.txt
  • Use wordlist with rules:
    john --wordlist=rockyou.txt --rules hashes.txt
  • Show cracked passwords:
    john --show hashes.txt
  • Run incremental brute-force:
    john --incremental hashes.txt
  • Mask attack (e.g. 8 characters, 3 digits last):
    john --mask='?a?a?a?a?a?a?d?d?d' hashes.txt
  • Use latest wordlists like SecLists’ passwords.
  • Start with wordlist + rules before brute force.
  • Use hash identification tool john --list=formats to find supported hashes.
  • Use GPU-accelerated tools like Hashcat for faster cracking.
  • Save and resume sessions with --session option.
