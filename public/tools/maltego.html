Maltego Complete Cheatsheet

Maltego is a powerful OSINT and link analysis tool used to discover relationships between people, domains, companies, emails, IPs, and more. It uses "transforms" to retrieve and correlate data in real-time.
  • Entities: Data points (e.g., person, email, domain)
  • Transforms: Actions performed on entities to gather data
  • Graphs: Visual representation of relationships
  • Machines: Automated sequence of transforms
EntityDescription
PersonFull name or alias
Email AddressTarget email accounts
Phone NumberLinked to individuals/companies
DomainInternet domain name
WebsiteSpecific URLs
IP AddressHost IPs for lookup
CompanyOrganization name
LocationGeo/physical data
  • [Domain] → DNS Name — Resolve to IPs, subdomains
  • [IP Address] → Netblock owner
  • [Email Address] → Domain — Related domain discovery
  • [Person] → Email addresses
  • [Website] → Hosted IP
  • [Domain] → Whois info
  • [Domain] → MX/NS records
  • Footprint L1/L2/L3 — Recon on a domain (passive to active)
  • Company Stalker — Gathers people & emails from a domain
  • Discover Social Networks
  • To WHOIS Info — For domain/IP ownership details
