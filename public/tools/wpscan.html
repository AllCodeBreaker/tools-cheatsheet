WPScan Complete Cheatsheet

  • Install WPScan:
    sudo gem install wpscan
  • Update WPScan:
    wpscan --update
  • Update Vulnerability Database:
    wpscan --update-vdb
  • Scan target:
    wpscan --url https://example.com
  • Enable verbose:
    -v
  • Output to file:
    --output results.txt
OptionDescription
--enumerate uEnumerate users
--enumerate pEnumerate plugins
--enumerate tEnumerate themes
--enumerate vpVulnerable plugins
--enumerate vtVulnerable themes
--enumerate cbConfig backups
  • Use basic auth:
    --http-user admin --http-pass password
  • Use API token:
    --api-token YOUR_TOKEN
  • Use --random-user-agent to evade basic filters
  • Combine user enumeration with other tools for deeper recon
  • Respect target’s rate limits and legal boundaries
  • Run as root or use sudo if needed for certain features
