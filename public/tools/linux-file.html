Linux Cheatsheet - File Management

Lists files and directories in the current directory.

OptionDescription
-lLong listing format
-aInclude hidden files
-hHuman-readable sizes
-RRecursive list

Example: ls -lah

Changes the current working directory.

Example: cd /var/log

Prints the current working directory.

Example: pwd

Output:
/home/john/projects

Creates a new directory.

OptionDescription
-pCreate parent directories as needed

Example: mkdir -p /tmp/new/project

Removes files or directories.

OptionDescription
-rRecursive delete (directories)
-fForce delete without prompt

Example: rm -rf /tmp/old

Warning: This command is irreversible!

Copies files or directories.

OptionDescription
-rCopy directories recursively
-uOnly copy updated files
-vVerbose output

Example: cp -ruv source/ backup/

Moves or renames files and directories.

Example: mv file.txt /home/user/docs/

Rename Example: mv oldname.txt newname.txt

Searches files and directories recursively.

ExampleDescription
find . -name "*.log"Find all .log files in current dir
find /etc -type f -size +1MFind files larger than 1MB
find . -mtime -7Files modified in last 7 days

Searches a prebuilt database for file names.

Example: locate passwd

Note: Run sudo updatedb to refresh database.

Displays a directory structure in a tree-like format.

Example: tree -L 2

Output:
./
├── bin
├── etc
└── usr

Displays detailed info about a file.

Example: stat myfile.txt

Output:
Size: 1024  Blocks: 8  Modify: 2023-08-01
