Bettercap Cheatsheet

  • Bettercap is a powerful, modular network attack and monitoring framework.
  • Supports MITM attacks, network sniffing, and more.
  • Works on multiple protocols like HTTP, HTTPS, DNS, and Wi-Fi.
CommandDescription
bettercap -iface <interface>Start Bettercap on a network interface
net.probe onEnable network host discovery
net.recon onEnable network reconnaissance
arp.spoof onStart ARP spoofing (MITM attack)
net.sniff onStart packet sniffing
  • arp.spoof: ARP spoofing for MITM attacks
  • net.sniff: Packet capture and analysis
  • http.proxy: HTTP proxying and injection
  • dns.spoof: DNS spoofing
  • wifi.deauth: Wi-Fi deauthentication attacks
  • Run Bettercap with root privileges.
  • Use in a controlled environment with permission.
  • Combine with Wireshark for deep packet analysis.
  • Customize modules via interactive CLI.
  • Keep Bettercap updated for latest features and fixes.
