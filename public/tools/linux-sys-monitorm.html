Linux Cheatsheet - System Monitoring & Performance

Interactive process viewer showing CPU, memory, and process usage.

top

Key commands inside top:

  • h: Help
  • k: Kill a process
  • q: Quit
  • 1: Toggle per CPU usage display

Report system performance metrics like processes, memory, paging, block IO, traps, CPU activity.

vmstat 2 5

Output shows every 2 seconds, 5 times.

Report CPU statistics and input/output statistics for devices and partitions.

iostat -xz 2 3

Options:

  • -x: Display extended statistics
  • -z: Skip devices with no activity
  • 2 3: Sample every 2 seconds, 3 times

Monitor individual tasks (processes) currently being managed by the kernel.

pidstat -u -r 2

Shows CPU (-u) and memory (-r) usage every 2 seconds.

Display amount of free and used memory in the system.

free -h

Options:

  • -h: Human-readable output (e.g., MB, GB)

Collect, report, or save system activity information.

sar -u 1 3

Reports CPU utilization every 1 second, 3 times.

  • htop: Interactive process viewer with more features than top
  • dstat: Versatile resource statistics
  • glances: Cross-platform system monitoring tool
  • nmon: Performance monitoring tool
