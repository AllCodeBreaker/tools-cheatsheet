Empire Cheatsheet

  • Empire is a post-exploitation framework focused on PowerShell and Python agents.
  • Supports various modules for lateral movement, persistence, and data exfiltration.
  • Used for penetration testing and red teaming.
CommandDescription
./empireLaunch Empire console
listenersManage listeners
usestager windows/launcher_batSelect stager to generate payload
usestager python/launcherGenerate Python stager
agentsList active agents
usemoduleSelect and run post-exploitation modules
  • Persistence: usemodule persistence/registry_run_key
  • Credential Dumping: usemodule credentials/mimikatz
  • Keylogging: usemodule collection/keylogger
  • Network Recon: usemodule discovery/arp_scanner
  • Credential Harvesting: usemodule collection/credentials/mimikatz
  • Keep Empire updated regularly for latest modules and security fixes.
  • Use encrypted communication channels for agent communication.
  • Test payloads in controlled environments before deployment.
  • Combine with other post-exploitation tools for better coverage.
