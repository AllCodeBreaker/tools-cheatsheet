CommandDescription
tail -f /var/log/syslogFollow system log in real-time
less /var/log/messagesView large log files with navigation
journalctlView systemd journal logs
journalctl -u nginx.serviceView logs of nginx service
dmesgPrint kernel ring buffer messages

Use grep or awk to filter logs:

grep "error" /var/log/syslog
journalctl | grep "fail"
awk '/pattern/ {print $0}' /var/log/messages
  • logrotate: Manage automatic log rotation and compression
  • sudo systemctl restart rsyslog: Restart syslog service
  • /etc/rsyslog.conf: Configuration file for syslog
