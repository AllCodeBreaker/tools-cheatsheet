Nikto Web Scanner Complete Cheatsheet

nikto -h <target_url_or_IP>

Scan target web server for common vulnerabilities and misconfigurations.

OptionDescription
-hHost to scan (domain or IP)
-pSpecify port(s), e.g. 80,443
-sslForce SSL mode (https)
-TuningEnable specific tests by category (e.g. 1 for file enumeration)
-PluginsSpecify plugins to run
-DisplayDisplay options (e.g., 123 for verbose, debug)
OptionDescription
-o <file>Save scan output to file
-Format <type>Output format: txt, csv, html, xml
-output <file>Alias for -o
  • Scan using custom user agent:
    nikto -h <host> -useragent "Mozilla/5.0"
  • Enable SSL scanning:
    nikto -h <host> -ssl
  • Use a proxy:
    nikto -h <host> -useproxy http://127.0.0.1:8080
  • Ignore specific tests:
    nikto -h <host> -skip test_id
  • Update Nikto plugins and database:
    nikto -update
  • Run as root or with elevated privileges for best results.
  • Combine Nikto with proxy tools like Burp Suite for deeper testing.
  • Use verbose output for debugging:
    nikto -h <host> -Display V
  • Keep Nikto updated regularly for new vulnerabilities and plugins.
