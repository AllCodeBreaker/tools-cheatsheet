Dirbuster Cheatsheet

  • DirBuster is a multi-threaded web application directory and file brute-forcer.
  • Used to discover hidden web directories and files.
  • Uses wordlists to guess directory and file names.
ActionDescription
Start DirBuster GUIRun dirbuster in terminal
Set Target URLEnter the base URL to scan
Select WordlistChoose wordlist from built-in or custom list
Set Number of ThreadsControl speed by adjusting threads
Start ScanBegin brute forcing directories/files
CommandDescription
dirbuster -u http://target/ -l wordlist.txt -t 50Scan URL with wordlist and 50 threads
dirbuster -u http://target/ -e php,html,txtSet file extensions to search for
  • Use large, relevant wordlists for better coverage.
  • Adjust thread count to avoid overwhelming the target.
  • Combine with proxy tools like Burp Suite for analysis.
  • Filter out common HTTP error codes in results.
  • Respect legal boundaries and get permission before scanning.
