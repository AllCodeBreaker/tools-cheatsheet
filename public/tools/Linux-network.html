Linux Cheatsheet - Networking

Display or manage network interfaces.

Modern: ip a or ip addr

Legacy: ifconfig (may require install)

Test connectivity to a host/IP.

Example: ping google.com

-c 4: Send 4 packets only
-i 0.5: Interval 0.5s

netstat: Show network connections (deprecated)

ss: Modern replacement

Example: ss -tuln (list listening TCP/UDP ports)

Query DNS records.

dig: Modern DNS tool
nslookup: Simpler but older

Example: dig openai.com

Install (Debian): sudo apt install dnsutils

Trace route of packets to destination (each hop).

Example: traceroute google.com

Install: sudo apt install traceroute

hostname: Show/set system hostname

nmcli: NetworkManager command-line tool

Example: nmcli dev status

Download or test network endpoints.

CommandDescription
wget URLDownload file
curl -I URLShow headers only
curl -O URLDownload as original name
